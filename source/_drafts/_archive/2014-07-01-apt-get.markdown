### 新增源
    $ sudo apt-add-repository ppa:nginx/stable
### 更新源
    $ apt-get update --fix-missing
### 卸载不再需要的依赖程序
    $ apt-get autoremove
### 使用 dpkg -i 安装软件 提示缺少依赖包时，执行
    $ apt-get -f install
