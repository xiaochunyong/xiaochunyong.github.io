* ⌘	Command 键
* ⌃	Control 键
* ⌥	Option 键
* ⇧	Shift 键
* ⇪	Caps Lock
* fn	功能键

* ⌘ D               Duplicate Line
* ⇧ Enter           Start New Line
* ⌥ ⌘ Enter        Start New Line Before Current
* ⌃ ⇧ J            Join Line
* ⌘ W               Select Word At Caret
* ⌘ ⇧ W           Unselect Word At Caret
* ⌘ Delete            Delete Line At Caret
* ⌘ fn Delete
* Column Selection Mode
